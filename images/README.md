# Images Folder

Bu klasöre görselleri ekleyiniz:

- **logo.png** - Site logosu (önerilen: 200x50 piksel)
- Diğer görseller ve resimler

## Logo Ekleme

Logo dosyanızı `logo.png` olarak bu klasöre ekleyin. HTML dosyalarında şu şekilde referans verilmiştir:

```html
<img src="images/logo.png" alt="Feza Savaş Logo">
```

veya sayfa klasöründen:

```html
<img src="../images/logo.png" alt="Feza Savaş Logo">
```
